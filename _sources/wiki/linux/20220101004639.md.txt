## ps


[^1]
[^1]:https://chenshinan.github.io/2018/02/24/Linux%E5%AD%A6%E4%B9%A0%EF%BC%9A%E5%9F%BA%E7%A1%80%E5%91%BD%E4%BB%A4%E5%8F%8A%E7%B3%BB%E7%BB%9F%E6%93%8D%E4%BD%9C/#more

```bash
ps
    # 查看进程信息

ps -a
    # 查看所有进程

ps -ef
    # 显示所有命令，连带命令行

ps -au
    # 显示进程较详细的资讯

```
