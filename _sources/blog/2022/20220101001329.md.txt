## 位运算


### 与

皆真即真

+ 三种结果
   + 1. 1 & 1 = 1
   + 2. 1 & 0 = 0
   + 3. 0 & 0 = 0

### 或

有真即真

+ 三种结果
   + 1. 1 | 1 = 1
   + 2. 1 | 0 = 1
   + 3. 0 | 0 = 0

### 异或

不同为真

+ 三种结果
   + 1. 1 | 1 = 0
   + 2. 1 ^ 0 = 1
   + 3. 0 ^ 0 = 0

### 取反

一元运算

有真即假  有假即真

+ 两种结果
    + 1. ~1 = 0
    + 2. ~0 = 1
