## Multiple return types in rust

```rust
fn main() {

enum Multiple {
A(u8),
B(char)
}


fn f(b: bool) -> Multiple {
if b {
Multiple::A(0)
} else {
Multiple::B('b')
}

}


println!("{:#?}",f(true));
println!("{:#?}",f(false));

}

```
