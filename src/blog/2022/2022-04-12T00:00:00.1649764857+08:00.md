## 人体

+ 男
    + 肩
        + 外肩稍宽于臀宽
    + 胸
        + 略宽于内肩
    + 腰
        + 稍微窄于胸宽
    + 臀
        + 上下宽度一致  类似横向长方形  
+ 女
    + 肩
        + 外肩略窄于臀宽
    + 胸
        + 略宽于内肩
    + 腰

    + 臀

### 透视

+ 视点 
    + 观察者眼睛所在的位置
+ 心点
    + 视点对画面的垂直落点, 即画面主视觉的焦点, 也是平行透视中的灭点
+ 余点
    + 成角透视中心点两侧的灭点
+ 天点
    + 竖直方向的顶灭点
+ 地点
    + 竖直方向的底灭点。
+ 视线 
    + 视点与被观察物体间的连线
+ 视中线
    + 视点与心点的连线, 是最短的一条中心视线
+ 水平线
    + 与地面平行朝向的直线
+ 竖直线
    + 与地面垂直朝向的直线
+ 视平线
    + 过心点的水平线, 代表视点的高度, 是上下分割画面的基准线
+ 正中线
    + 过心点的竖直线, 与视平线互相垂直, 是左右分割画面的基准线
+ 地平线
    + 视域最远处天空与地（海）面的交界线, 平视时与视平线重合

### 一点透视

+ 近大远小的规律
+ 近处斜棱要向上延伸
+ 远处斜棱要向下延伸


### 绘画方式

#### 斜二测画法

先画出 x 和 y 轴 <br />
之后以任意一轴为底 <br />
由 (0, 0) 为起始点 <br />
按 45° 角的大小延伸出 z 轴 <br />


视角67度, 视距为心点到画幅最远角的1.5倍；

视角60度, 视距为心点到画幅最远角的1.7倍；

视角53度, 视距为心点到画幅最远角的2.0倍。

视角在90度～67度之间, 称为特近视距；

视角在67度～53度之间, 称为近视距；

视角在53度～37度之间, 称为中视距；

视角小于37度, 称为远视距。

通常作图, 均采用2倍视距构图, 便于计算, 也可以采用1.5倍视距, 便于作图。

因为实际作图时, 画纸都不会很大, 而透视图往往需要很大的空间, 主体物则会显得很小, 因此视距如果越近的话, 透视图就能够画得越大, 便于分析和标注复杂的透视关系。
