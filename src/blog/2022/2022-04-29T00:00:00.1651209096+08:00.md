## 将正在运行的任务脱离终端运行


```bash
sleep 60
    # 按 ctrl-z 暂停任务
bg
    # 让其转入后台继续运行
jobs -l
    # 获取 jobid
disown -h %<jobid>
    # 让其脱离终端运行
disown -a
    # 让多个进程脱离终端运行
```


```bash
kill -s SIGTSTP pid
    # 暂停进程
kill -s SIGCONT pid
    # 放入后台运行
```
