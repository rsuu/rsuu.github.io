## ls

```bash
ls 
    # 列出文件和目录

ls -l
    # 列出文件信息

ls -l -a
    # 显示隐藏文件

ls -LR
    # 递归显示文件

```

### 格式

–: 一个标准文件
d: 一个目录
l: 一个符号链接
c: 字符设备（将数据作为字节流，例如终端）
b: 块设备（以块的方式处理数据，例如存储设备

读（r）、写（w）、执行（x） 权限
文件的所有者、文件的所有组、和其它的用户
