    TPT¶

    TPT[The Powder Toy] 是一个可以组合各种元素的沙盒游戏

    使用¶

    由于性能问题 一般不会在游戏内制作过大的东西

    元素¶

    • 墙[Walls]

      • 所有的墙都无法被摧毁

    • 电路元件[Electronics]

    • 可控材料[Powered Materials]

    • 传感器[Sensors]

    • 爆炸物[Force]

    • 动力材料[Explosives]

    • 气体[Gasses]

    • 液体[Liquids]

    • 粉末[Powders]

    • 固体[Solids]

    • 放射性元素[Radioactive]

    • 特殊元素[Special]

    • 生命游戏[Life]

    • 工具[Tools]

    气体¶

    Powders :: 粉末¶

    粉末是会被引力影响的可移动不粘着的粒子

    • DUST :: 尘埃

      • 非常轻

      • 可燃

      • 火柴人默认发射 DUST

    • STNE :: 石头

      • 重粒子

      • 可熔化

    • SNOW :: 雪

      • 轻粒子

      • 当 ICE 在压力下破碎后产生

      • 加热后转化为 WATR

    • CNCT :: 混凝土

      • 重粒子

      • 比石头更结实

    • SALT :: 盐

      • 可溶于水

    • BRMT :: 破碎金属

      • 当铁被腐蚀或在压力下破碎而形成

    • SAND :: 沙子

      • 重粒子

      • 熔化为玻璃

      • 在 1700°C 时可熔化成玻璃

    • BGLA :: 破碎玻璃

      • 重粒子

      • 当玻璃在压力下破碎形成

      • 可熔化

    • YEST :: 酵母

      • ~37°C 时生长

    • FSEP :: 引信粉末

      • 燃烧缓慢

    • BCOL :: 破碎煤

      • 重粒子

      • 燃烧缓慢

    • FRZZ :: 冷凝子

      • 轻粒子

      • 融化时形成冰

      • 在正常的水中扩散

    • GRAV :: 引力子

      • 非常轻的尘埃

      • 根据速度变换颜色

      • 无视引力

    • ANAR :: 反空气粒子

      • 非常轻的尘埃

      • 向引力和压力相反的方向移动

      • 在绝对零度时燃烧

    • PQRT :: 破碎石英

      • 石英的破碎状态

    • BREL :: 破碎电子元件

      • 由 EMP 爆炸形成

      • 在压力下持续通电后会转变为 EXOT

    • CLST :: 粘土

      • 与水混合后产生胶

    • SAWD :: 木屑

      • 可浮于水上

    Liquids :: 液体¶

    • 水

      • 可导电

      • 凝固

      • 灭火

    • 石油

      • 可燃

      • 在低压或高温下转化为 GA

      • 可以由 NEUT 和 NITR 形成

    • 岩浆

      • 点燃可燃物

      • 当金属或其它物质熔化时产生

      • 冷却时凝固

    • 酸

      • 溶解几乎所有东西

    • 蒸馏水

      • 不导电

    • 盐水

      • 导电

      • 不容易凝固

    • 熔蜡

      • 液态的蜡

      • 在45度时凝固成 WAX

    • 液氮

      • 超低温

      • 在接触到任何热的东西后消失

    • 柴油

      • 液体柴油

      • 在高温或高压下爆炸

    • 液氧

      • 超低温

      • 会与火发生反应

    • 光晕

      • 在压力下发出微光

    • 苏打水

      • 缓慢释放 CO2

    • 反常物

      • 与正常状态相矛盾

      • 用装饰色绘制其他元素

    • 浆糊

      • 胶质物

      • 在压力下硬化

    • 胶

      • 粘度可变的液体

      • 可导热

    • 肥皂

      • 产生气泡

      • 洗掉装饰色

      • 消灭病毒

    • 水银

      • 体积随温度变化

      • 可传电

    • 病毒

      • 使所有接触的东西转变为病毒T

      • 能量类型的元素不会受影响

    ¶

    • GOO :: 粘性物

      • 在压力下变形或消失

    • ICE :: 冰

      • 在压力下粉碎

      • 使空气变冷。

    • WOOD :: 木头

      • 可燃.

    • PLNT :: 植物

      • 吸水生长

    • BMTL :: 可破碎金属

      • 可导电的建筑材料

      • 可熔化

      • 在压力下会破碎

    • WAX :: 蜡

      • 在一定的温度下融化。

    • GLAS :: 玻璃

      • 可熔化

      • 在压力下破碎反射光子.

    • NICE :: 氮冰

      • 非常冷

      • 轻微加热后可熔化成 LN2

    • COAL :: 煤

      • 燃烧缓慢

      • 变热后变红

    • BRCK :: 砖

      • 可破坏的建筑材料

    • FUSE :: 引信

      • 燃烧缓慢

      • 可被引燃

    • IRON :: 铁

      • 被盐水腐蚀

      • 可以用于向 WATR 导电

    • DRIC :: 干冰

      • 由 CO2 冷却后产生

    • SPNG :: 海绵

      • 吸水

      • 不可移动的固体

    • RIME :: 固体

      • 由蒸汽极速冷却凝华而成

    • VINE :: 藤

      • 可沿 WOOD 生长

    • SHLD :: 壳

      • 通电时生长

    • FILT :: 光子过滤器

      • 改变光线的颜色

    • QRTZ :: 石英

      • 可破碎材料

      • 导电

      • 在低温下变脆

    • TTAN :: 钛

      • 比所有其它金属熔点都要高, 可阻挡一切压力.

    • GOLD :: 金

      • 耐腐蚀金属

      • 可以逆转铁的腐蚀

    • CRMC :: 陶磁

      • 在压力下变结实

    • HEAC :: 快速导热体

      • 快速导热
